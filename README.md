A fully functional C++ console-based Library Management System that simulates real-world library operations including login, book management, issue/return system, due date tracking, and fine calculation using file handling.

✅ Current Working Version
Complete system runs in terminal (C++)
Admin and User login
Add, search, update, delete books
Issue and return books
Due date and fine calculation
Data stored using text files
Proper record management
🚧 Frontend (Work in Progress)

This repository also contains HTML, CSS, and JavaScript files intended for future web integration.
Currently, these files are not fully connected to the C++ backend.

💻 Technologies Used
C++
File Handling
OOP Concepts
HTML / CSS / JavaScript (planned integration)
 How to Run

Compile and run the C++ file in terminal:

g++ main.cpp -o main
./main
 Note-

The core logic and complete functionality are implemented in C++.
Web version integration is under development.
